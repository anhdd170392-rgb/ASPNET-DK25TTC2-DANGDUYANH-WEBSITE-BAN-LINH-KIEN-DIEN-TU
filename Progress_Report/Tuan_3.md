# BÁO CÁO TIẾN ĐỘ ĐỒ ÁN - TUẦN 3

- **Đề tài:** Xây dựng website bán laptop sử dụng ASP.NET MVC 5 và SQL Server
- **Thời gian thực hiện:** 06/07/2026 - 12/07/2026
- **Trọng tâm tuần:** Hiện thực hóa code chức năng cốt lõi (Phía Khách hàng)

---

## Nội dung công việc
Cấu hình kết nối DB trong Web.config. Tiến hành viết code cho phía Khách hàng: Màn hình trang chủ, danh sách sản phẩm, bộ lọc theo Hãng/Nhu cầu. Xây dựng logic Giỏ hàng (thêm, sửa số lượng, xóa) và Đặt hàng trực tuyến, Xem lịch sử mua hàng.

## Tài liệu liên quan
Tài liệu kỹ thuật ASP.NET Routing, ActionResults, Session trong ASP.NET để lưu trữ giỏ hàng tạm thời.

## Khó khăn gặp phải
Xử lý đồng bộ dữ liệu giỏ hàng khi người dùng thay đổi số lượng trực tiếp tại giao diện front-end mà không làm tải lại toàn bộ trang (cần áp dụng AJAX/jQuery).

## Kết quả đạt được
Vận hành ổn định luồng mua hàng từ xem sản phẩm đến lưu đơn hàng vào database.
