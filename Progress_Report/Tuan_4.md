# BÁO CÁO TIẾN ĐỘ ĐỒ ÁN - TUẦ4

- **Đề tài:** Xây dựng website bán laptop sử dụng ASP.NET MVC 5 và SQL Server
- **Thời gian thực hiện:** 13/07/2026 - 19/07/2026
- **Trọng tâm tuần:** Hiện thực hóa code chức năng quản trị (Phía Admin) và bổ sung tính năng cải tiến

---

## Nội dung công việc
Tích hợp giao diện quản trị SB Admin 2. Viết chức năng CRUD (Thêm, Sửa, Xóa, Xem) cho các thực thể: Quản lý sản phẩm (Laptop), Quản lý thể loại, Quản lý đơn hàng, Quản lý bài viết tin tức. Thiết lập giao diện Quản lý vai trò và phân quyền.

## Tài liệu liên quan
ASP.NET Identity và Phân quyền dựa trên Roles (Admin/Khách hàng).

## Khó khăn khi viết thêm chức năng
Gặp trở ngại lớn khi tích hợp thêm các chức năng nâng cao theo đề xuất cải tiến như: Thiết lập tính năng thông báo nổi khi có đơn hàng mới; Gặp khó khăn khi viết logic gợi ý thông minh tự động kiểm tra trùng khớp thông tin khách hàng cũ để gợi ý tạo mới profile khách hàng tại trang Admin; Thử nghiệm cấu hình Realtime làm tốn nhiều thời gian cấu hình.

## Kết quả đạt được
Hoàn thiện toàn bộ các màn hình CRUD của Admin và tích hợp thành công phần phân quyền cơ bản.
