# BÁO CÁO TIẾN ĐỘ ĐỒ ÁN - TUẦN 1

- **Đề tài:** Xây dựng website bán laptop sử dụng ASP.NET MVC 5 và SQL Server
- **Thời gian thực hiện:** 22/06/2026 - 28/06/2026
- **Trọng tâm tuần:** Giai đoạn chuẩn bị và nghiên cứu lý thuyết cơ sở

---

## Nội dung công việc
Xác định lý do chọn đề tài, mục tiêu, đối tượng và phạm vi nghiên cứu. Tìm hiểu tổng quan về framework ASP.NET MVC 5, giao thức HTTP, các ngôn ngữ bổ trợ như HTML, CSS, JavaScript, thư viện jQuery và Bootstrap.

## Tài liệu liên quan
Sách chuyên ngành, khóa học lập trình ASP.NET MVC (TEDU), mã nguồn giao diện Template Bootstrap SB Admin 2.

## Khó khăn gặp phải
Hệ thống hóa lại kiến thức về mô hình MVC (Model-View-Controller) và cách nhúng các thư viện front-end sao cho tối ưu, chuẩn bị môi trường cài đặt .NET Framework.
